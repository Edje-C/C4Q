There are 3 type of list:
definition list
which show terms and definitions
unordered list
which show bullets
ordered list
which show numbers and letters

Unordered list can have a type of attributewhich can be applied to the UL or LI tag. The 3 values are:

  • type = "square"
  • type = "disc"
  • type = "circle"

Ordered list can have a type attribute too.Used along with a start attributey cna show:

  1. lower case letters
  2. upper case letters
  3. lower case roman numerals
  4. upper case roman numerals
  5. numbers
